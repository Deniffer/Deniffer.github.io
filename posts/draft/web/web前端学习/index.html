What is HTML （HyperText Markup Language）

学习前端的时候，我会问自己一个问题，

  • 什么是HTML？

    • 在我看来，HTML只不过是一堆有着不同用途的tag组成的标记语言，浏览器的解释器通过解析tag，来渲染出对应的画面。举个简单的例子

      1
2
3

      <html>
		<h1>Hello World!</h1>
</html>

      浏览器会根据h1的tag 把Hello World认为是一级标题，对他进行加粗，仅此而已。 那么也就说，我们写HTML的意图就是在于让解释器根据我们的这一些tag去渲染我们的html文件，然后就会显示我们想要看到的画面。 所以说我们的html 是写给浏览器看的，浏览器必须得明白。因此html就是一种语言，一种我们和浏览器沟通的桥梁。

    • 课程老师对HTML的解释：The language your web browser uses to describe the content and structure of web pages

  • HTML语法的构成

    • Point of View ： HTML is just Nested Box ！

    我们知道HTML是什么之后，很简单，我们就需要学习和浏览器沟通的方式，也就是HTML的语法。（就像你为了和外国人沟通要学英语是一个道理），所谓的Opening Tag 和Closing Tag 就是表示tag的作用域。
