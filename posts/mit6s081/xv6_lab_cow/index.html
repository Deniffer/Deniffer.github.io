Demand paging

Only load the segments that we need to run the program, until the time we need a specific segment,that occurs a page fault and we start to load the specific segment so that the program can continue to run .

  • 其实demand paging的好处就类似于延迟加载，也就说我们点击一个program的时候，只加载我们当前所需要用到的段，这样做的好处是我们可以快速的启动一个program，很多工作会被延迟到真正需要的时候，因此给用户的感觉还是挺不错的，也就是说本来很多需要在start up需要被做的事情可以平均分配到一些时间段去做 或者不需要做 这样的方式可以给用户更好的体验，同时也节省了os的部分内存。(参考绝地求生，8G的内存也可以run，但是会卡顿，原因就是因为页频繁的被换进换出导致的)
